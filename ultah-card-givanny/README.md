# Simple-Web-Design - Ultah Card Givanny

Halaman ucapan ulang tahun yang indah dan interaktif untuk istri tercinta.

## ✨ Fitur Utama

- 🎆 **Fireworks Animation** - Animasi kembang api saat halaman dimuat
- 🎉 **Confetti** - Potongan kertas jatuh dari atas
- ✨ **Cursor Spark** - Percikan cahaya mengikuti cursor
- 📸 **Photo Popup** - Klik foto untuk melihat dalam ukuran besar
- 📱 **Mobile-Friendly** - Responsive design untuk semua ukuran layar
- 🎨 **Modern UI** - Glass effect dan warna-warna pastel yang elegan
- 🖼️ **Fixed Background** - Background mengikuti konten sampai bawah

## 🚀 Cara Mengakses

1. **Direct Path**: `ultah-card-givanny/index.html`
2. **GitHub Pages**: Aktifkan di settings repository
3. **Local**: Buka file `index.html` dengan browser

## 📁 Struktur File

```
ultah-card-givanny/
├── index.html                 # Halaman utama
├── styles.css                 # Styling dan animasi
├── background_ultah.jpg       # Background image
├── foto-istri-single.jpg      # Foto saat single
└── foto-istri-marriage.jpg    # Foto saat menikah
```

## 🎨 Fitur Teknis

- **Responsive Design**: Mobile-first approach
- **CSS Animations**: Smooth fireworks, confetti, dan spark effects
- **Click-to-Expand**: Foto hanya expand saat diklik (bukan hover)
- **Fixed Background**: Background tetap di viewport saat scroll
- **Zero Dependencies**: Pure HTML, CSS, dan vanilla JavaScript

## 💝 Customization

Untuk menyesuaikan:
- Edit teks di `index.html`
- Ganti foto dengan nama file yang sama
- Ubah warna di CSS variables di awal `styles.css`
